# Custom fonts

Drop font files into this folder and they are **auto-detected** by the builder —
they appear in the text tool's font picker under the **Custom** group. No
restart or code change needed; the list refreshes when the builder loads.

Supported formats: `.ttf`, `.otf`, `.woff`, `.woff2`

The font family name shown in the picker is the file name without its
extension (e.g. `AKAGEEHtrial.ttf` → "AKAGEEHtrial").
